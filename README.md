## About

This website presents my academic profile, research interests, publications, teaching experience, education, credentials, professional experience, and academic activities.

I am a PhD Candidate in Computer Science at Iowa State University and a Graduate Student Contributor at Ames National Laboratory. My work spans theoretical computer science, molecular programming, chemical reaction networks, analog computing, theoretical AI, optimization, machine learning, and scientific discovery.

## Deployment

This website is deployed using GitHub Pages.

The repository should be named:

```text
Sapi98.github.io
```

GitHub Pages automatically serves:

```text
index.html
```

as the homepage at:

```text
https://Sapi98.github.io/
```

---

## Updating the Website

After editing the website files:

```bash
git add .
git commit -m "Update academic website"
git push
```

GitHub Pages will automatically publish the updated site.

---

## Future Page Updates

When adding a new public page:

```text
1. Add the page link to the navigation bar.
2. Add page-specific SEO metadata in the page head.
3. Add the page URL to sitemap.xml.
4. Test the page in both light and dark mode.
5. Check mobile responsiveness.
```

For pages that are still under development, keep them out of the navigation and sitemap until ready.
